## üîß If Something Goes Wrong

### Page doesn't load
```bash
npm run build  # Rebuild
npm run dev    # Test locally at http://localhost:3000/about
```

### Styling looks wrong
- Check browser cache (Ctrl+Shift+Del)
- Clear .next folder: `rm -r .next`
- Rebuild: `npm run build`

### Links don't work
- Verify contact email addresses are correct
- Test links in browser console
- Check internal routes (/shop, /contact, /guides)

### Need to rollback
```bash
git revert <commit-hash>  # Undo changes
git push origin main       # Push revert
```
