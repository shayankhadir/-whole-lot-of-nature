## Product Description Optimization Guidelines

To improve rankings, product descriptions should follow this structure:

1.  **H1 Title:** Include the primary keyword (e.g., "Premium Organic Potting Mix - 5kg").
2.  **Opening Paragraph:** clearly state what the product is and its main benefit, using the primary keyword within the first 100 words.
3.  **Features List:** Use bullet points for readability (e.g., "100% Organic", "Odorless", "Rich in NPK").
4.  **Benefits Section:** Explain *why* it matters (e.g., "Promotes faster root growth", "Retains moisture longer").
5.  **Care Instructions:** (For plants) Include light, water, and soil requirements.
6.  **FAQ Schema:** Answer common questions like "Is this safe for pets?" or "How often should I use this?".

## Technical SEO Recommendations

*   **Meta Titles:** Keep under 60 characters. Format: `[Product Name] | [Main Benefit] | Whole Lot of Nature`
*   **Meta Descriptions:** Keep under 160 characters. Include a call to action (e.g., "Shop now for free shipping!").
*   **Image Alt Text:** Describe the image using keywords (e.g., "bag of organic potting soil on white background").
*   **URL Structure:** Clean, descriptive URLs (e.g., `/shop/organic-potting-mix-5kg` instead of `/shop/product-123`).
*   **Internal Linking:** Link to related products (e.g., "Works best with our [Neem Oil]").
